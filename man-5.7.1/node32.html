Building with the MSVC++ compiler in MSys2

It is possible to perform at automated build of SYMPHONY using the MSVC++ compiler cl with GNU autotools in the MSys2 environment. To do so, follow the instuctions for building in Unix-like environments (see Section 2.2.2), except when configuring, use the command 

 ./coinbrew fetch build SYMPHONY@\VER --enable-msvc
