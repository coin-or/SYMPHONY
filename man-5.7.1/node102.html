Microsoft Visual C++

First, download SYMPHONY-5.7.1 and unpack the archive if it is required. You now have three options. You can either compile on the command-line using the automated DEVENV build system or NMAKE utility or you can use the provided project and solution files. For all of the following options, first go to the SYMPHONY Applications USER MSVisualStudio v8 directory.
