Sub Menu Browsing

SYMPHONY's interactive optimizer also allows the user to reach the lower level menu commands from the higher level menus. In other words, the user has the flexibility to use submenu commands without entering the corresponding submenu. As an instance, all three of the following sessions have the same result: 

  •     SYMPHONY: display parameter verbosity

  •     SYMPHONY: display
    Please type 'help'/'?' to see the display options!
    SYMPHONY\Display: parameter verbosity

  •     SYMPHONY: display
    Please type 'help'/'?' to see the display options!
    SYMPHONY\Display: parameter
    Please type 'help'/'?' to see the list of available parameters!
    SYMPHONY\Display\Parameter: verbosity
This flexibility is also enabled for the load command and the Set submenu. The followings are all valid commands: 
    SYMPHONY: load sample.mps

    SYMPHONY: load sample.mod sample.dat

    SYMPHONY: set
    SYMPHONY\Set: verbosity 3

    SYMPHONY: set verbosity 3
    SYMPHONY: set param_file my_param
