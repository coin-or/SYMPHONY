                       FILE MENU

  The file menu can also be invoked by pressing Alt-f.

  Load from file:  
	Load a graph from a file. See the example file 
	dantzig.graph for the input file format.

  Save to file:
	Save a graph into a file. 

  Save PostScript:
	Save the postscript format of the graph into a file.
	If the viewable part of the graph is too big to fit 
        on an 8x11 paper, it will be scaled down and/or 
        rotated appropriately. 

  Quit from application:
	Quit from the entire application.
